## 四、Axios 请求封装验证

### 4.1 请求拦截器

#### TC-FE-AXIOS-REQ-001 Token 注入

- **前置条件**：localStorage 中存有 token
- **操作**：发起任意 API 请求
- **预期结果**：
  - 请求头自动携带 `Authorization: Bearer {token}`

#### TC-FE-AXIOS-REQ-002 无 Token 时不注入

- **前置条件**：localStorage 中无 token
- **操作**：发起任意 API 请求
- **预期结果**：
  - 请求头不含 Authorization 字段

### 4.2 响应拦截器

#### TC-FE-AXIOS-RES-001 成功响应解析

- **操作**：后端返回 `{ code: 200, message: "success", data: [...] }`
- **预期结果**：
  - Axios 拦截器提取 `response.data.data` 返回给调用方
  - 即业务代码拿到的是 data 部分

#### TC-FE-AXIOS-RES-002 业务错误提示

- **操作**：后端返回 `{ code: 404, message: "资源不存在" }`
- **预期结果**：
  - ElMessage.error 弹出错误提示 "资源不存在"
  - Promise reject，业务代码进入 catch

#### TC-FE-AXIOS-RES-003 网络错误提示

- **操作**：断开后端服务，发起请求
- **预期结果**：
  - ElMessage.error 弹出 "网络异常"
  - Promise reject

#### TC-FE-AXIOS-RES-004 请求超时

- **操作**：后端响应超过 10 秒
- **预期结果**：
  - 触发超时错误
  - ElMessage.error 弹出错误提示
